# Changelog

## [1.2.0] - 2024-01-01
### Added
- Complete Movie Request System
- `/request` command with natural language support
- `/myrequests` command for users
- `/pendingrequests` command for admins
- Inline moderation buttons
- Bulk approve/reject actions
- Auto-approve when movie added to CSV
- Session support for custom rejection reasons
- User notification system
- Request statistics

### Enhanced
- Updated Dockerfile with session support
- Improved .htaccess with security headers
- Enhanced error logging
- Better admin interface
- Improved user experience

## [1.1.0] - 2023-12-01
### Added
- Multi-channel support
- CSV-based movie database
- Smart search with caching
- Pagination for movie browsing
- Channel type detection (Public/Private)
- User points system
- Admin statistics
- Render.com deployment support

## [1.0.0] - 2023-11-01
### Initial Release
- Basic movie search functionality
- Single channel support
- Simple CSV management
- Basic Telegram bot structure
